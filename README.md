# Charades – Информационно-игровое приложение «Генератор шарад»

## Описание проекта

**Charades** — это настольное приложение, разработанное на платформе **WPF (.NET Framework)**, предназначенное для игровой генерации шарад различной сложности.

Пользователь угадывает слово по набору изображений, получает очки и соревнуется в таблице лидеров.

Приложение включает:
- систему регистрации и авторизации пользователей
- игровой процесс с уровнями сложности
- подсказки и систему штрафных очков
- таблицу лидеров
- административную панель для управления пользователями и шарадами

## Функционал

### Пользователь:
- регистрация и вход в систему
- запуск игры
- выбор сложности (Easy / Medium / Hard)
- получение шарад с изображениями
- ввод ответа
- система подсказок (-5 очков)
- подсчёт очков и статистики

### Администратор:
- добавление и удаление пользователей
- добавление и удаление шарад
- создание новых шарад с загрузкой изображений
- управление базой данных (JSON)

## Структура хранения данных

Данные сохраняются локально в формате JSON:

- `Data/users.json` — пользователи
- `Data/charades.json` — список шарад

Изображения:
- `Images/` — локальная папка хранения картинок

## Технологии

- C#
- WPF (.NET Framework)

- Windows Forms (элементы админ-панели)

## Установка и запуск

### Путь:
1. Запустить файл `setup.exe`
2. Следовать инструкциям установщика
3. Запустить приложение через ярлык

## Особенности проекта

- автономная работа без сервера
- локальное хранение данных
- система уровней сложности
- динамическая загрузка изображений
- защита от повторного использования подсказки
- расширяемая архитектура
